# Telegram Bot — Інструкція деплою на Railway

## Файли в цій папці
- `bot4.py` — код бота
- `requirements.txt` — залежності Python
- `Procfile` — інструкція для Railway як запускати бота
- `*.ogg` — голосові файли

---

## Деплой на Railway (безкоштовно)

### 1. Отримай новий токен
Зайди в Telegram → @BotFather → /mybots → вибери бота → API Token
(старий токен що був у коді вже скомпрометований — скинь його через /revoke)

### 2. Залий файли на GitHub
1. Зайди на https://github.com → New repository
2. Назви, наприклад: `my-tg-bot`
3. Залий всі файли з цієї папки

### 3. Задеплой на Railway
1. Зайди на https://railway.app
2. Увійди через GitHub
3. New Project → Deploy from GitHub repo → вибери репо
4. Зачекай поки збудується

### 4. Додай токен як змінну середовища
1. У Railway → твій проект → вкладка **Variables**
2. Натисни **New Variable**
3. Name: `BOT_TOKEN`
4. Value: твій токен від BotFather
5. Натисни **Add**

Бот автоматично перезапуститься і запрацює!

---

## Якщо щось пішло не так
- Перевір вкладку **Logs** в Railway — там видно помилки
- Переконайся що токен правильний
